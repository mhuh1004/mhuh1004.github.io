Arduino Waterproof Thermometer and Data Analysis Using MATLAB Project

For the class IDEA215: Introduction to Sensors, Measurement, and Data Analysis taught by professor Daniel Moller at Wesleyan University, we were tasked with creating a waterproof heat measuring device using a thermometer, a bread board, wires and an Arduino. We were then asked to be able to take the data recorded from the thermometer in a cup of water overnight to create a linear regression equation of the trajectory of the temperature using MATLAB data analysis.
